## Description
Professional AI prompt generator helps users create efficient, precise AI prompts. Built-in frameworks and templates boost AI output quality by 10 times.

## Usage
```bash
# Generate prompt
/ai-prompt-generator --task "Write a blog" --model "gpt-4" --style "detailed"

# Use template
/ai-prompt-generator --template "cot" --topic "data analysis"

# Optimize existing prompts
/ai-prompt-generator --action "optimize" --input "your-prompt.txt"
```

### Parameter Explanation
- `--task`: Task description
- `--model`: Target AI model (gpt-4, claude, gemini, qwen)
- `--style`: Prompt style (detailed, concise, creative, analytical)
- `--template`: Use template (cot, role, few-shot, chain)
- `--action`: Operation type (generate, optimize, test)
